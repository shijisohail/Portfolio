# Sharjeel Sohail — Portfolio

Personal portfolio website for Sharjeel Sohail, Senior Backend Engineer.

**Live:** https://shijisohail.github.io/Portfolio/
**Repo:** https://github.com/shijisohail/Portfolio

> GitHub Pages is not yet enabled. To activate: repo Settings → Pages → Source: `main` branch → `/` (root).

## Tech

Single-file static HTML/CSS/JS — no build step, no dependencies, no framework.

- Fonts: Inter + Fira Code via Google Fonts
- Icons: Font Awesome 6
- Deployed via: any static host (GitHub Pages, Vercel, Netlify, etc.)

## Structure

```
Portfolio/
└── index.html   # entire site — styles, markup, and scripts in one file
```

## Sections

| Section | ID | Description |
|---|---|---|
| Hero | — | Title, summary, KPI counters, CTA buttons |
| About / Skills | `#skills` | Skill cards by category |
| Experience | `#experience` | Timeline of roles at Uforia, Merik, HyperNym, CRCC |
| Projects | `#projects` | Filterable project cards |
| Achievements | `#achievements` | Key impact cards with before/after metrics |
| Contact | `#contact` | Contact info + message form |

## Running locally

```bash
# open directly in browser
open index.html

# or serve with any static server
npx serve .
python3 -m http.server 8080
```
